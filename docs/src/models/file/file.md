# File Model

| Field Name | Type     | Description                                                                                |
| ---------- | -------- | ------------------------------------------------------------------------------------------ |
| `id`       | `string` | Unique Endpoint ID or file name in case of config parsing <br>**Generates by mock-server** |
| `name`     | `string` | Provided file name                                                                         |
| `length`   | `number` | Size of file in bytes                                                                      |
